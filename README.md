# OnlineTraining

### **NodeJS version: v18.14.0**

### **NPM version: 9.3.1**

### **MySQL version: 8.0.32**

### how to use the project:

### With front-end

### 1. `git clone project`

### 2. `cd ./front-end`

### 3. `npm install`

### 4. `npm start`

### With back-end

### 1. `git clone project`

### 2. `cd ./back-end`

### 3. `npm install`

### 4. `Create .env variable and add data`

### 5. `cd ./back-end ==> run cmd create table: npx sequelize-cli db:migrate`

### 6. `npm start`
